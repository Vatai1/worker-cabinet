Vacation Transfer Application Template (DOCX)
==============================================

Generated via: POST /api/vacation/generate-transfer-application
Template purpose: vacation_transfer_template

Template Placeholders
---------------------

Root level:
  {last_name} {first_name} {middle_name}  — full name
  {short_name}                              — Фамилия И.О.
  {position}                                — job title
  {department}                              — department name
  {year}                                    — year
  {date_today}                              — current date (ДД.ММ.ГГГГ)

Loop {#transfers} ... {/transfers}:
  {original_start}   — original vacation start date
  {original_days}    — original vacation duration in days
  {new_start}        — new requested start date
  {new_days}         — new requested duration in days
  {delta_direction}  — "увеличив" or "сократив" (increased/decreased)
  {delta_days}       — absolute difference in days
  {note}             — reason text (if provided)

                               {date_today}

                               Подпись: ____________ / {short_name} /

Generation Notes
-----------------
- Only fetches transfers with 'approved' status
- Dates formatted as ДД.ММ.ГГГГ (Russian locale)
- Template uploaded via admin panel → Documents → Templates
